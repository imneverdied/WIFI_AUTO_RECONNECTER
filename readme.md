WIFI自動重連
===========
2021/8/4 add Python.ver </br>

原理為 PING GOOGLE 成功時顯示 網路連接正常，失敗時呼叫WIFI.bat重新連接<br>
注意: WIFI.bat 內設定的WIFI名稱必須為已知WIFI(連上過的WIFI) <br>

使用方法 <br>
第一次使用時請先用TXT編輯WIFI.bat內的"WIFI的名子"改成自己要連的WIFI名子<br>
點擊 WIFI.exe  <br>
